# k0wl0n Homebrew Tap

This repository contains [Homebrew](https://brew.sh/) formulae for open source tools maintained by [k0wl0n](https://github.com/k0wl0n).

## Available Formulae

| Tool | Description |
| ---- | ----------- |
| **[gctx](https://github.com/k0wl0n/gctx)** | A powerful CLI to manage and switch between multiple Google Cloud Platform (GCP) accounts and ADC credentials seamlessly. |

## Usage

### 1. Add the Tap
First, add this tap to your Homebrew installation:

```bash
brew tap k0wl0n/tap
```

### 2. Install Tools

To install **gctx**:
```bash
brew install gctx
```

## Updating

To get the latest versions of the tools:

```bash
brew update
brew upgrade
```

## Contributing & Issues

*   If you find an issue with a specific tool (e.g., `gctx`), please report it in that tool's repository.
*   If the issue is with the installation/formula itself, feel free to open an issue here.
